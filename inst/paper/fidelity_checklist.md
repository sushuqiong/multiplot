## Instructions for manual verification

For each screenshot, please compare against the ggmultiplot output generated by:

```r
library(ggplot2)
library(ggmultiplot)

# Bar chart
ggplot(mpg, aes(class, hwy)) +
  stat_summary(geom = "col", fun = "mean", aes(fill = class)) +
  stat_summary(geom = "errorbar", fun.data = "mean_se", width = 0.2) +
  ggchoice("prism")  # or "spss" or "matlab"

# Boxplot
ggplot(mpg, aes(class, hwy)) +
  geom_boxplot(aes(fill = class)) +
  ggchoice("prism")  # or "spss" or "matlab"
```

Please note any discrepancies in:
1. Font rendering (family, weight, size hierarchy)
2. Colour matching (bar/box fill)
3. Spacing/padding around plot elements
4. Axis tick mark appearance
5. Any other visible difference

This will be used to compute a fidelity score for the paper.
